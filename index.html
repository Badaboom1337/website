Git Linux

Git — система контролю версій

Системи контролю версій (VCS) — це програмні інструменти, які дозволяють відслідковувати зміни у файлах, керувати версіями коду та спільною роботою. Найпопулярнішою є Git.

Переваги Git

  • Розподілена система — кожен розробник має повну копію репозиторію.
  • Висока швидкість роботи.
  • Підтримка гілок (branching) для паралельної розробки.
  • Надійність та безпека даних.
  • Широке використання у світі (GitHub, GitLab, Bitbucket).

10 базових команд Git

  1. git init — створення нового репозиторію.
  2. git clone [url] — клонування існуючого репозиторію.
  3. git status — перевірка стану файлів.
  4. git add [файл] — додавання файлу у staging.
  5. git commit -m "повідомлення" — збереження змін.
  6. git log — історія комітів.
  7. git branch — перегляд імен гілок.
  8. git checkout [гілка] — перемикання між гілками.
  9. git merge [гілка] — злиття гілок.
  10. git push / git pull — відправка чи отримання змін з віддаленого репозиторію.

GNU/Linux

Коротка інформація

GNU/Linux — це сімейство операційних систем з відкритим кодом, побудованих на ядрі Linux. Вона широко використовується у серверах, десктопах, мобільних пристроях та вбудованих системах. Linux відомий своєю стабільністю, безпекою та гнучкістю.

Найбільш популярні дистрибутиви

  • Ubuntu — простий у використанні дистрибутив для новачків.
  • Debian — стабільний і надійний.
  • Fedora — сучасний, з новими технологіями.
  • Arch Linux — мінімалістичний, для досвідчених користувачів.
  • CentOS / Rocky Linux — серверні рішення.

Команди Linux CLI

  • pwd — показати поточну директорію.
  • echo "Hello" — вивести текст.
  • ls — показати список файлів.
  • cd [шлях] — змінити директорію.
  • touch file.txt — створити файл.
  • mkdir newdir — створити папку.
  • cp file1 file2 — копіювати файл.
  • mv file1 file2 — перемістити або перейменувати файл.
  • rm file.txt — видалити файл.
  • history — історія виконаних команд.

Bash-скрипт (імітація авторизації)

#!/bin/bash
echo "Введіть логін:"
read login
echo "Введіть пароль:"
read -s password

if [[ $login == "admin" && $password == "1234" ]]; then
  echo "Авторизація успішна! Ласкаво просимо, $login."
else
  echo "Помилка авторизації. Спробуйте ще раз."
fi
